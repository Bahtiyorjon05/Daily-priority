# Focus Music Folder

Place your focus/study music files here (.mp3, .wav, .ogg formats supported).

## Recommended Music Types:
- Lo-fi beats
- Ambient music
- Classical piano
- Nature sounds
- White noise

## File Naming:
Name your files clearly, for example:
- lofi-study-beats.mp3
- rain-sounds.mp3
- classical-piano.mp3

The app will automatically detect and play these files during focus sessions.
